- Créer un compte Heroku
- Télécharger et installer Heroku CLI
- > heroku login
- Créer une app sur l'interface web de Heroku
- linker votre repo avec l'app avec la commande heroku git:remote -a *heroku application-name*
- Créer un fichier Procfile
- Créer un fichier requirements.txt
- push votre code sur heroku avec la commande > git push heroku master
